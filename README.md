# Trump
A command line utility written in rust that makes your files great again!

![Alt text](http://cdn.meme.am/instances/500x/64741784.jpg)  
## Commands:
###Help:
  
    trump --help
  
###Version:
  
    trump -v
  
###Files:
  
    trump <File>
  
###Directories:
  
    trump -r <Directory>
